## GET\_ALPHA\_LOOKUP

Command that returns the index of the first item in the list that begins with the specified character.


### Name

`GET_ALPHA_LOOKUP ()`


| Parameter | Type | Description           |
| --------- | ---- | --------------------- |
| LETTER    | STR  | Letter used for Index |
| ROOMID    | INT  | Room ID value         |


### SendToProxy

`<alpha_lookup>`
