## has\_skip\_fwd

Media Player supports forward skipping through media.

### Signature

`<has_skip_fwd></has_skip_fwd>`


| Parameter | Description |
| --- | --- |
| bool | true/false |


### Returns

`None`


### Example

```xml
<capabilities>
   <has_skip_fwd>true</has_skip_fwd>
</capabilities>
```
